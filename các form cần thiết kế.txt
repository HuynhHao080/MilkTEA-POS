=====================================================
HƯỚNG DẪN CODE FORM MỚI
=====================================================

1. Tạo form mới:
   - Add → Windows Form
   - Đặt tên: frm<Tên>.cs (ví dụ: frmProducts.cs)
   
2. Cấu trúc code (theo frmCategories):
   - #region Constructor & Initialization
   - #region Data Loading & Display
   - #region Event Handlers
   - #region Form Data Management
   - #region Image Handling (nếu có)
   - #region Search Functionality (nếu có)
   - #region Toolbar Actions
   - #region Database Operations
   - #region Helper Methods
   
3. Checklist khi làm form:
   [ ] Load data
   [ ] Hiển thị DataGridView
   [ ] Thêm mới
   [ ] Sửa
   [ ] Xóa (có confirm)
   [ ] Tìm kiếm
   [ ] Validation
   [ ] Error handling
   [ ] Loading indicator
   [ ] Security (nếu upload file)
   
4. Security checklist:
   [ ] Path traversal validation
   [ ] File size limit (nếu upload)
   [ ] Extension whitelist (nếu upload)
   [ ] SQL injection safe (dùng EF Core)
   [ ] Null checks
   
5. Update README.md:
   - Thêm form vào danh sách
   - Note features đã làm
   - Note issues (nếu có)

=====================================================
QUY ƯỚC CODE
=====================================================

Đặt tên:
- Form: frm<Tên>.cs
- Controls: <Type><Tên>
  + Button: btnAdd, btnEdit, btnDelete
  + TextBox: txtName, txtDescription
  + Label: lblTitle, lblSearch
  + DataGridView: dgvCategories, dgvProducts
  + Panel: pnlHeader, pnlMain, pnlForm
  + PictureBox: picPreview
  
Code style:
- Dùng async/await cho DB calls
- Try-catch với friendly messages
- Validation trước khi save
- Dùng #region để organize
- Comments khi cần thiết
