🔐 PASSWORD SECURITY (BCrypt) - PRODUCTION READY
=====================================================

✅ Implemented:
- BCrypt.Net-Next 4.1.0
- Work factor: 12 (4096 iterations)
- Auto salt per password
- Constant-time comparison
- OWASP compliant

✅ Migration Strategy:
- ✅ Dual column: password_hash + password (legacy)
- ✅ Auto-migrate: Plain text → BCrypt (first login)
- ✅ Plain-text storage REMOVED from new code
- ✅ Only password_hash is updated now

✅ Login Flow:
1. Check password_hash (BCrypt)
2. If exists → BCrypt.Verify()
3. If not → Check password (plain text - legacy)
4. If match → Auto hash → Save → Success!

✅ Change Password Flow:
1. Verify current (BCrypt)
2. Hash new password (BCrypt work=12)
3. Update password_hash (BCrypt only)
4. No plain-text storage
