=====================================================
🔐 PASSWORD SECURITY (BCrypt)
=====================================================

✅ Implemented:
- BCrypt.Net-Next 4.1.0
- Work factor: 12 (4096 iterations)
- Auto salt per password
- Constant-time comparison
- OWASP compliant

✅ Migration Strategy:
- Dual column: password_hash + password
- Auto-migrate: Plain text → BCrypt (lần login đầu)
- password column sẽ xóa sau này

✅ Login Flow:
1. Check password_hash (BCrypt)
2. If exists → BCrypt.Verify()
3. If not → Check password (plain text)
4. If match → Auto hash → Save → Success!

✅ Change Password Flow:
1. Verify current (BCrypt)
2. Hash new password (BCrypt work=12)
3. Update password_hash (BCrypt)
4. Update password (plain text)
5. Save to DB

📝 HƯỚNG DẪN CODE FORM MỚI
=====================================================

1. Tạo form mới:
   - Add → Windows Form
   - Đặt tên: frm<Tên>.cs (ví dụ: frmProducts.cs)
   
2. Cấu trúc code (theo frmCategories):
   #region Constructor & Initialization
   #region Data Loading & Display
   #region Event Handlers
   #region Form Data Management
   #region Image Handling (nếu có)
   #region Search Functionality (nếu có)
   #region Toolbar Actions
   #region Database Operations
   #region Helper Methods
   
3. Checklist khi làm form:
   [ ] Load data
   [ ] Hiển thị DataGridView
   [ ] Thêm mới
   [ ] Sửa
   [ ] Xóa (có confirm)
   [ ] Tìm kiếm
   [ ] Validation
   [ ] Error handling
   [ ] Loading indicator
   [ ] Security (nếu upload file)
   
4. Security checklist:
   [ ] Path traversal validation
   [ ] File size limit (nếu upload)
   [ ] Extension whitelist (nếu upload)
   [ ] SQL injection safe (dùng EF Core)
   [ ] Null checks
   
5. Update file này:
   - Đánh dấu ✅ khi hoàn thành
   - Note features đã làm
   - Note issues (nếu có)
